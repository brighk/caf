Using GPUs on SCIAMA

See Submitting Jobs on SCIAMA if you are not already familiar with how to do this.

Using GPUs interactively

There are two 40GB GPU nodes: gpu01 and gpu02, with 128 cores each. To use them interactively, type:

srun --pty --mem=1g --gres=gpu:1 -J interactive -p gpu.q /bin/bash

and to check which gpu:

nvidia-smi
echo &CUDA_VISIBLE_DEVICES

Submitting batch GPU jobs

To run batch jobs on GPU nodes, ensure your job submission script includes a request for GPUs:

#!/bin/bash
#SBATCH --partition=gpu.q
#SBATCH --gres=gpu

Only 1 job can run on a GPU at a time, so upto 4 jobs can run concurrently on the gpu.q.

Nvidia Multi-Instance GPU (MIG)

The nvidia GPUs can be partitioned into smaller instances to allow for better resource allocation and testing, you can read more about MIG on the nvida website. GPU01 and GPU02 nodes have 2x 40GB GPUs that have been partitioned as:
